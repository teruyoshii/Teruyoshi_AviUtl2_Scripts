## 既知のバグ

- 簡易シェーダーが全般的にうまく動かない (現状開発途中で投げ出しているので当然ではある)
- パスグラデーションと段階グラデーションにおいて、開始色透明度や終了色透明度を99から100に変更すると急激に色が変化してしまう
- 一括イージングにおいて、登録の方をちゃんと全フレーム再生してからでないと正しく反映されない
- 一括イージングにおいて、イージングを編集してからシークバーを動かすと、一瞬だけ編集前のパラメータのものが表示されることがある
- 一括イージングを外部スクリプトに適応しようとした際、正しく配置していても「一括イージング登録が見つかりません」というエラーが発生する
